# Tools Directory

This directory contains utility scripts and tools for project maintenance.

## Auto-Commit Tools

- `auto-commit.sh` - Automated git commit script
- `manage-auto-commit.sh` - Management script for auto-commit functionality
- `auto-commit.log` - Auto-commit execution log
- `auto-commit.pid` - Process ID file for auto-commit
